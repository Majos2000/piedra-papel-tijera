# Piedra, Papel y Tijera

Implementación en Python del clásico juego de Piedra, Papel y Tijera. Se juega por consola contra la computadora.

## Requisitos

- Python 3 instalado.

## Como ejecutar

Abrir una terminal en la carpeta del proyecto y ejecutar:

    python main.py

## Como se juega

1. El programa muestra las tres opciones (1: Piedra, 2: Papel, 3: Tijera).
2. El usuario escribe el numero correspondiente a su elección.
3. La computadora elige una opcion al azar.
4. Se muestra el resultado de la ronda.
5. El programa pregunta si desea jugar de nuevo. Si la respuesta es "si" inicia una nueva ronda; si es "no" termina el juego.
